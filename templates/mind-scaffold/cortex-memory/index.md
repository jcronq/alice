# cortex-memory

Alice's groomed knowledge base. Atomic, wikilinked markdown organized
by note type. Single source of truth for durable facts.

## Layout

| Folder | Type | Purpose |
|---|---|---|
| `people/` | Entity | One note per person (the owner, trusted contacts) |
| `projects/` | Entity | One note per active project / system |
| `reference/` | Reference | How-things-work docs, design records, policies |
| `feedback/` | Feedback | Behavioral rules extracted from user feedback |
| `sources/` | Source | External materials (papers, articles, third-party data) |
| `dailies/` | Daily | Chronological activity log, one file per day |
| `research/` | Research | Investigation artifacts and synthesis notes |
| `conflicts/` | Conflict | Active contradictions between vault notes |
| `decisions/` | ADR | Architectural decision records |
| `findings/` | Finding | Hypothesis test results from investigations |

Full taxonomy with graph colors: [[memory-types]].
Layout policy and grooming conventions: [[memory-layout]].
