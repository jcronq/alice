# Long-Term Memory

Curated, durable memories. Alice reads this in main sessions per
`AGENTS.md`'s session-startup procedure. It's intentionally separate
from `memory/YYYY-MM-DD.md` (raw daily logs) and `cortex-memory/`
(groomed wiki) — this file is the small, distilled essence Alice
wants top-of-mind every session.

Add entries as they accrue. Sample shape:

## Owner

- (preferences, communication style, quirks, things Alice should remember)

## (Other principals)

- (per-person notes for trusted contacts Alice talks to)

## Lessons

- (recurring-mistake fixes, behavioral corrections that earned a
  durable rule rather than a transient correction)
